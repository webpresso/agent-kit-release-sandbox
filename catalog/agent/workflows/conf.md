# /conf: The Confidence Protocol

Goal: gather requirements with zero ambiguity and low cognitive load.

## Steps

1. **Prep (silent)**: read AGENTS.md + relevant files; infer defaults from existing patterns.
2. **Gap analysis**: identify unknowns vs. safe assumptions.
3. **Interview loop** (one question at a time):
   - Context → Question → Default → Optional alternatives.
4. **Validate**: confirm answers are precise, secure, and schema-aligned.
5. **Contract**: present final summary and request explicit approval.

Checklist: one question at a time, sensible defaults, no secrets in chat, remind about docs updates.
