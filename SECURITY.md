# Security Policy

## Supported versions

Security fixes are targeted at the latest published `@webpresso/agent-kit` release. If you are running an older version, upgrade and retest before filing a report.

## Reporting a vulnerability

Please do **not** open a public issue for suspected vulnerabilities.

Use GitHub private vulnerability reporting for this repository:

- <https://github.com/webpresso/agent-kit/security/advisories/new>

If you cannot access that flow, open a minimal issue at <https://github.com/webpresso/agent-kit/issues> requesting a private follow-up, and do not include exploit details, secrets, or proof-of-concept payloads in public.
