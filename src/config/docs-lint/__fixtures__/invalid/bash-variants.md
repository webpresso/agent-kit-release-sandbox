# Guide with Bash Language Variants

This document has deprecated commands in different bash language labels, which should ALL be flagged.

## Bash Variant

```bash
just lint-file src/a.ts
```

## Sh Variant

```sh
pnpm vitest src/b.test.ts
```

## Shell Variant

```shell
just typecheck cli2
```

## Zsh Variant

```zsh
just test file src/c.test.ts
```

All bash language variants should be checked for deprecated commands.
